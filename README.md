# ControleAcademico
 Controle Academico de uma Universidade

 O Controle Academico tem:
- Curso
- Professor
- Turma
- Alunos

 Requisitos do sistema:
- Cadastrar um curso
- Cadastrar um professor
- Cadastrar uma turma
- Matricular alguns alunos
- Todos os alunos
- Todos os alunos de uma turma
- Professor da turma
